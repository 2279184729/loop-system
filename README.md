# Claude Code 父子多层嵌套自适应Loop系统

## 全栈高阶多Agent方案 - 完整实现

### 🎯 核心能力

| 特性 | 说明 |
|------|------|
| **自适应双模式切换** | 简单任务极速直出 / 复杂任务多层嵌套Loop调度 |
| **多层嵌套Loop** | 外层父Agent全局迭代 + 内层多子Agent并行自修复 |
| **分布式子Agent调度** | 独立进程、独立工作区、独立上下文、并行执行 |
| **轮次熔断机制** | 父Agent最大5轮方案规划 / 子Agent最大15轮自循环 |
| **自适应节流** | 简单任务零子进程，极致节省CPU和内存 |
| **隔离内网适配** | 仅依赖Anthropic API，完全适配隔离网络环境 |

### 📁 项目结构

```
loop-system/
├── orchestrator.py       # 父调度Agent核心引擎（大脑）
├── child_agent.py        # 子执行Agent模块（工人）
├── config.py             # 全局配置
├── utils.py              # 工具函数库
├── run_demo.py           # 完整演示脚本
├── run_demo.ps1          # Windows启动脚本
├── run_demo.sh           # Linux/Mac启动脚本
├── demo_project/         # 演示项目
│   ├── backend/          # 后端模块
│   ├── frontend/         # 前端模块
│   └── database/         # 数据库模块
├── workspaces/           # 子Agent独立工作区
└── logs/                 # 运行日志
```

### 🚀 快速启动

#### Windows
```powershell
# 运行完整演示
.\run_demo.ps1 -Demo

# 执行指定任务
.\run_demo.ps1 -Task "修复登录接口空指针bug"
.\run_demo.ps1 -Task "从零搭建全栈用户管理系统"
```

#### Linux/Mac
```bash
# 运行完整演示
./run_demo.sh --demo

# 执行指定任务
./run_demo.sh --task "修复登录接口空指针bug"
./run_demo.sh --task "从零搭建全栈用户管理系统"
```

#### 直接使用Python
```bash
# 完整演示
python run_demo.py

# 简单任务
python orchestrator.py "修复登录接口空指针bug"

# 复杂任务
python orchestrator.py "从零搭建全栈用户管理系统"
```

### 🔄 工作流程

```
用户输入任务
    │
    ▼
┌─────────────────┐
│  阶段0: 复杂度判定 │  ← 自适应关键词分析 + 启发式规则
└────────┬────────┘
         │
    ┌────┴────┐
    │         │
    ▼         ▼
┌───────┐ ┌───────────────────────┐
│简单任务│ │      复杂任务            │
│极速直出│ │  阶段1: 全局方案规划Loop  │
│       │ │  阶段2: 子Agent调度Loop  │
│0子进程 │ │  阶段3: 结果回流汇总Loop  │
│       │ │  阶段4: 全局自检修复Loop  │
└───────┘ └───────────────────────┘
    │         │
    └────┬────┘
         ▼
    ┌─────────┐
    │ 任务闭环 │
    └─────────┘
```

### 🛡️ 防失控机制

- **进程隔离**: 子Agent崩溃不影响父调度
- **轮次熔断**: 父5轮/子15轮，杜绝无限循环
- **自动回收**: 任务完成自动终止子进程
- **网络适配**: 完全禁用外网搜索和下载
